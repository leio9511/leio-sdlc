# PR-001: Implement Fundamental Data Fetcher via Akshare

## 1. Objective
Implement a robust function to fetch real-time fundamental data (PE and Total Market Cap) for A-shares using `akshare`, ensuring authentic financial metrics can be obtained.

## 2. Scope & Implementation Details
- Target Project: `/root/.openclaw/workspace/AMS`
- `adapter.py` (or `finance_fetcher.py`): Create a `fetch_fundamental_data()` function that calls a stable `akshare` endpoint (like `stock_a_indicator_lg` or equivalent) to get real `市盈率-动态` (PE-TTM) and `总市值` (Market Cap) for stocks. It must return a pandas DataFrame structured for merging.
- `tests/test_data_source.py`: Add a test that calls `fetch_fundamental_data()` and asserts that the returned DataFrame contains the columns `代码`, `市盈率-动态`, and `总市值`, and that the PE values are not uniformly hardcoded (e.g., checking that at least two different stocks have different PE values).

## 3. TDD & Acceptance Criteria
- Test Script: `tests/test_data_source.py`
- The test must explicitly fetch fundamental data and verify that `市盈率-动态` is present, numeric, and varies across different stock codes.
- `pytest tests/test_data_source.py` must pass.
