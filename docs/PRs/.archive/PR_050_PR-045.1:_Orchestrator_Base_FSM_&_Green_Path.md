# PR-045.1: Orchestrator Base FSM & Green Path

## 1. Description
Implement the foundational Finite State Machine (FSM) for `orchestrator.py` covering States 1 (Polling), 2 (Branching), 6 (Green Path Merge), and 7 (Fault Recovery).

## 2. Requirements
- Poll for PRs with `status: open` and mark them `in_progress`.
- Idempotent git branch creation and checkout (`feature/PR_XXX`).
- Call `merge_code.py` and delete branch upon success.
- Update PR status to `closed` after merge.
- Implement fault recovery on script startup to resume `in_progress` PRs.
